# YDOM for VS Code - v1.0.0 Inicial Release 🚀

## Resumen

La primera versión oficial de soporte estructurado para **YDOM (YAML DOM)** ya está disponible en tu editor. Una gramática enfocada 100% en legibilidad, velocidad y semántica, olvidando los falsos atajos de marcado tradicional.

## Características y Soporte

- 🎉 **Lanzamiento de Soporte Nativo**: Implementación pura de gramática usando JSON TextMate.
- 🏷️ **Key-Tagging Inteligente**: Color azul clásico para nodos y etiquetas basados en indentación estricta de YAML.
- ⚡ **Control Flow y Eventos (YDOM Specs)**: Resaltado agresivo brillante para los comandos de inyección lógica como eventos nativos (`@click`, `@submit.prevent`).
- 🔗 **Data Binding Sensible**: Parseo de inyección de propiedades tipo prop-drilling (`:value`).
- 🔤 **Manejo de Textos**: Interpretación sintáctica doble; para comillas clásicas y cadenas multilínea con interpolación (` ` `).
- 🧩 **Pares Ágiles**: Auto-matching rápido y cerrado para paréntesis estructurales `()`, comillas dobles y backticks de contenido dinámico.
