# YDOM Support for VS Code

Este es el soporte de lenguaje oficial para **YDOM** (YAML DOM).
Añade reglas personalizadas de sintaxis y autocompletado avanzado que no están presentes en YAML nativo.

## Funciones

- Resaltado jerárquico impecable de `Etiquetas:` basándose en identación.
- Colores específicos diseñados a medida para **Atributos nativos**, **Eventos UI (@)** y **Data Binding (:)**.
- Autocierre y navegación de pares de acentos graves `` ` `` y paréntesis `()`.
